# GB10 (DGX Spark, aarch64, compute capability 12.1) build.
# Replaces requirements/cuda.txt's torch/flashinfer pins with real wheel
# dependencies from personal GitHub Releases, since no official aarch64/sm_121
# wheels exist upstream yet. Modeled on requirements/xpu.txt's own
# `@ URL` GitHub-Release-wheel pattern (vllm_xpu_kernels).
#
# flash-attention is NOT listed here -- it's consumed via CMake at build time
# (see cmake/external_projects/vllm_flash_attn.cmake, VLLM_FLASH_ATTN_PREBUILT_PKG
# path) rather than as a normal pip dependency, since vLLM's own
# vllm.vllm_flash_attn package nests the compiled extension as a sibling file
# via a relative import (`from . import _vllm_fa2_C`) that a top-level pip
# package can't satisfy directly. build_gb10.sh installs it explicitly before
# the vLLM build so the CMake step can find it via `import vllm_flash_attn`.
